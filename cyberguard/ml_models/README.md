# ml_models/

This directory is auto-populated when you run:

    python setup_models.py

Generated files:
  - sms_model.pkl        : Multinomial Naive Bayes (SMS spam)
  - sms_vectorizer.pkl   : TF-IDF vectorizer for SMS
  - sms_model_lr.pkl     : Logistic Regression (comparison model)
  - sms_model_meta.txt   : Accuracy / AUC metadata
  - url_model.pkl        : Random Forest (URL classification)
  - url_model_meta.txt   : Accuracy / feature importances
